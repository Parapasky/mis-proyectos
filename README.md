# Mis Apuntes de Git 🚀

En este repositorio estoy aprendiendo a usar los comandos básicos de Git para gestionar mis proyectos.

## 1. Comandos Básicos de Git


| Comando         | Definición Breve                                 | Ejemplo de Uso          |
| :---            | :---                                             | :---                    |
| git --version   | Muestra la versión instalada de Git              | git --version           |
| git init        | Inicializa un repositorio en la carpeta actual   | git init                |
| git add .       | Agrega todos los cambios al área de preparación  | git add .               |
| git commit      | Registra los cambios con un mensaje descriptivo  | git commit -m "mensaje" |
| git status      | Muestra el estado actual del repositorio         | git status              |

---
> *Tip de Oro:* El flujo de trabajo ideal siempre es: 
> git add .  ➜  git commit  ➜  git status
